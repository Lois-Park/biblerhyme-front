### **바이블라임(BibleRhyme) 화면 정의서 (v1.1: 검색 기능 추가)**

| **MAIN-01** | 메인 (최신 음원 목록) | 서비스의 첫 화면. 기본적으로 최신순으로 정렬된 음원 목록을 보여준다. | 헤더(검색창 포함), 최신/베스트 탭, 음원 카드 목록, 미니 플레이어 |
| **MAIN-02** | 메인 (베스트 음원 목록) | '좋아요' 또는 특정 로직에 따라 산정된 인기 음원 순위 목록을 보여준다. | 헤더(검색창 포함), 최신/베스트 탭, **랭킹**이 표시된 음원 카드 목록, 미니 플레이어 |
| **SONG-01** | 음원 상세 | 특정 음원을 클릭했을 때 진입하는 화면. 댓글 등 상세 정보를 확인한다. | 음원 정보(제목, 생성자), 가사 표시 영역, **소셜 기능(좋아요/싫어요/공감/공유)**, **댓글 목록**, 댓글 입력창 |
| **AUTH-01** | 로그인 | 소셜 로그인 또는 이메일 로그인을 통해 사용자를 인증한다. | 로고, 소셜 로그인 버튼(구글, 카카오 등), 이메일/비밀번호 입력창, 회원가입/비밀번호 찾기 링크 |
| **AUTH-02** | 회원가입 | 신규 사용자가 서비스에 가입한다. | 약관 동의, 이메일/비밀번호/닉네임 입력창 |
| **UGC-01** | 음원 생성 | 사용자가 직접 AI 음원을 생성하는 페이지. | **보컬 프롬프트 입력창**, **비트 프롬프트 입력창**, 생성하기 버튼, 생성 진행 상태 표시 영역 |
| **MY-01** | 나의 라이브러리 | 내가 생성했거나 '좋아요'를 누른 음원 목록을 보여준다. | 헤더, 내가 생성한 음원 탭, 좋아요한 음원 탭, 음원 카드 목록 |
| **SEARCH-01** | 검색 결과 | 헤더의 검색창을 통해 검색을 실행한 결과 목록을 보여준다. | 헤더, 'OO'에 대한 검색 결과, 검색된 음원 카드 목록 |
