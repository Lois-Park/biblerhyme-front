### **바이블라임 프로젝트 - 1주차 개발 핵심 내용 (2025.11.18)**

**1. 개발 환경 및 데이터베이스 구축**

*   **기존 Next.js 프로젝트 분석:** 제공된 Next.js 프로젝트의 구조(App Router, `pnpm` 사용) 및 의존성(`package.json`을 통해 MUI 라이브러리 확인)을 분석했습니다.
*   **Prisma ORM 도입:** 데이터베이스 관리를 위해 Prisma ORM을 설치하고 `dotenv`를 설정했습니다.
*   **데이터베이스 스키마 정의:** SQLite를 기반으로 `User`, `Song`, `Social`, `Comment` 모델에 대한 데이터베이스 스키마를 설계하고 초기 마이그레이션을 성공적으로 완료했습니다.
*   **Prisma Client 인스턴스:** 애플리케이션 전역에서 사용할 Prisma Client 인스턴스를 생성하여 데이터베이스 접근을 용이하게 했습니다.

**2. 백엔드 - 로그인/회원가입 API 개발**

*   **인증 라이브러리 설치:** `next-auth` 라이브러리를 설치하고 기본 설정을 완료했습니다.
*   **비밀번호 해싱:** `bcryptjs`를 사용하여 사용자 비밀번호를 안전하게 해싱하는 기능을 구현했습니다.
*   **인증 프로바이더 설정:** 이메일/비밀번호 기반 인증을 위한 `CredentialsProvider`를 `next-auth`에 설정했습니다.
*   **`next-auth` API 라우트:** `src/app/api/auth/[...nextauth]/route.ts` 경로에 `next-auth` 동적 API 라우트를 구현하여 로그인 및 세션 관리를 처리합니다.
*   **회원가입 API 엔드포인트:** `src/app/api/auth/signup/route.ts` 경로에 새로운 사용자 등록을 처리하는 회원가입 API 엔드포인트 구현했습니다.
*   **환경 변수 설정:** `NEXTAUTH_SECRET` 환경 변수 설정.
